## 一、使用說明書 (User Guide)
本系統提供基礎與進階房貸分析，主打現代化的操作體驗與視覺化呈現。操作流程如下：

1. **基礎參數輸入**：
   - 於左側面板手動輸入「房屋總價」、「自備款」、「年利率」、「貸款年限」與「寬限期」。
   - *(請在此處插入：整體介面與輸入面板截圖)*

2. **區域預設與快速估價**：
   - 點開左側的「區域預設」，可選取台灣各縣市及行政區，系統會自動套用該區域的大樓/公寓/透天均價與參考利率。
   - 提供坪數估價計算，方便快速試算。
   - *(請在此處插入：區域預設下拉選單與估價功能截圖)*

3. **執行試算與動態視覺化圖表**：
   - 點擊「開始試算」後，右側畫面的數字會以動畫形式遞增，總結每月還款、總利息等關鍵數據。
   - 中央顯示本息比例的圓餅圖，下方則繪製「本金餘額趨勢圖」（支援滑鼠滾輪縮放區間，滑鼠移過可檢視單期詳細數據）。
   - *(請在此處插入：動態折線圖與圓餅圖結果截圖)*

4. **進階分析模組工具列**：
   - 畫面上方的工具列提供了專業的財務與資安分析。包含：
     - **RL 尋優**：利用強化學習 (Q-Learning) 找出最佳的提前還款時機。
     - **ZKP 認證**：零知識證明展示，不洩漏真實薪水即完成財力驗證。
     - **壓力測試**：蒙地卡羅 (Monte Carlo) 利率壓測與 GBM 資產水下風險評估。
   - *(請在此處插入：進階工具列與 AI 分析文字報告截圖)*

5. **情境比較與專業報表匯出**：
   - 可將當前結果加入「📌 情境比較」頁籤，系統會自動繪製雷達圖與熱力矩陣，最多可比較 3 套方案。
   - 支援將報表匯出成 PDF（可切換學術/商務版型）及 CSV 檔（具備 SHA256 防篡改驗證）。
   - *(請在此處插入：情境比較雷達圖與 PDF 匯出結果截圖)*

## 二、問題排除與學習心得 (Troubleshooting & Reflections)

### 1. 開發瓶頸與問題排除
- **圖表渲染與 UI 效能**：初期引入大量攤還期數（例如 40 年共 480 期）與多條輔助線時，自己繪製的 Chart 出現了閃爍與延遲。
  - **解決方案**：改用 `DoubleBuffered` 機制，並透過 `GraphicsPath` 與 `SmoothingMode.AntiAlias` 進行 GDI+ 繪圖優化，同時加入計時器 (Timer) 控制動畫更新頻率，確保整體流暢度。
- **進階數學模型整合防呆**：實作強化學習 (Q-Learning) 與零知識證明 (ZKP) 時，涉及了大量的迭代運算與大數運算 (ModPow)；若使用者未先試算就點擊，容易造成崩潰 (Crash) 或無窮迴圈。
  - **解決方案**：獨立拆分演算法至副程式，補齊例外處理 (try-catch)，並加入防呆條件（如檢視 `totalLoan > 0`），確保程式健全性。

### 2. 學習心得
- **GUI 設計體會**：跳脫傳統 WinForms 灰灰的刻板印象，學到了如何自己刻畫扁平化按鈕 (Flat UI)、自訂無邊框標題列、實作色彩主題切換 (明亮/夜間) ，以及如何利用 ToolTip 與明顯的提示標籤取代容易被忽略的系統報錯，大幅提升 UX。
- **除錯技巧**：透過 Visual Studio 的逐步執行與變數監視，釐清了在計算本息平均攤還時，各期本金與利息之間因浮點數 (double) 所造成的微小小數點誤差，並修正了最後一期的結算機制。
- **GitHub 操作**：熟悉了 Git 的版控流程，了解頻繁地 Commit 與推送 (Push) 原碼的好處。尤其在專案後期加入大型功能 (如 PDF 匯出引擎) 時，能夠放心地進行重構，這對於程式碼的維護非常有幫助。

## 三、公式與資料來源 (Formulas & Data Sources)

### 1. 核心計算公式
- **本息平均攤還公式**：
  $$ 每月還款金額 = 貸款本金 \times \frac{月利率 \times (1+月利率)^{還款總月數}}{(1+月利率)^{還款總月數} - 1} $$
- **強化學習 (Q-Learning) 貝爾曼方程式更新律**：
  $$ Q(s,a) \leftarrow Q(s,a) + \alpha \left[ R + \gamma \max_{a'} Q(s', a') - Q(s, a) \right] $$
- **零知識證明 (ZKP) 驗證核心** (Fiat-Shamir Heuristic)：
  基於離散對數問題，驗證條件為： $g^s \pmod p = t \times C^c \pmod p$

### 2. 參考資料與數據來源
- **縣市房價與利率數據**：參考「內政部不動產資訊平台」與「金融監督管理委員會」近期之公開統計資料（系統內所用之單價與利率為模擬示意值）。
- **學術與理論模型**：
  - 現金流貼現 (DCF)、淨現值 (NPV) 與內部報酬率 (IRR)：參考基礎財務管理文獻。
  - 幾何布朗運動 (GBM) 與 Cox-Ingersoll-Ross (CIR) 模型：參考金融工程相關理論。
- **PDF 輸出排版**：採用 IEEE 學術論文雙欄式標準格式作為產生樣板來源。
