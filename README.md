# 房貸計算器（WinForms / .NET Framework 4.7.2）

房貸試算與報告輸出工具，支援區域估價、攤還表、情境比較、壓力測試與 PDF 匯出。

## 主要功能 (Top 20 Features)

1. 房貸基準試算（本息平均攤還）
2. 寬限期彈性設定與跨期現金流演算
3. 年提前還款模擬策略
4. 區域預設大數據估價（分縣市/行政區/建物類型）
5. 行政區智能自動篩選與快速搜尋
6. 視覺化本金餘額與占比曲線（互動縮放圖表）
7. 三維度情境比較矩陣（手動新增/快速生成多套方案）
8. PDF 嚴謹學術與商務報告一鍵生成（原生位元組寫入引擎）
9. 利率壓測敏感度矩陣（正負利率變動現金流壓力）
10. **蒙地卡羅 (Monte Carlo / CIR) 利率隨機漫步壓測**
11. **幾何布朗運動 (GBM) 房產溺水負淨值風險模型**
12. 理論投資分析（NPV淨現值 / 機會成本評估）
13. 現代租買決策數學模型（DCF現金流貼現資產交叉點）
14. 不動產投報率深度運算（內部報酬率 IRR、資本化率 Cap Rate）
15. **Merkle Hash Tree 區塊鏈資料完整性防篡改技術**
16. **霍夫曼無損資料壓縮演算法 (Huffman Coding) 快取體積最佳化**
17. Reward-based 強度學習風險評分建議引擎
18. Tornado 單因子風險敏感度圖像分析（Tornado Chart）
19. Explainability Card 財務專家可解釋建議卡
20. 現代化響應式 UI（自訂色彩主題、平滑動畫縮放、抗鋸齒字體）

## 執行環境

- Visual Studio Community 2026
- .NET Framework 4.7.2
- C# 7.3

## 如何執行

1. 使用 Visual Studio 開啟方案。
2. 設定啟動專案為 `1113354_陳冠瑋_房貸計算器`。
3. 按 `F5` 執行。

## 使用流程

1. 輸入房價、自備款、利率、年限、寬限期。
2. （可選）使用區域預設與行政區搜尋快速估價。
3. 按 `開始試算`。
4. 可使用 `壓測` / `隨機壓測` 比較風險。
5. 以 `PDF` 匯出完整報告。
