# 房貸計算器（WinForms / .NET Framework 4.7.2）

房貸試算與報告輸出工具，支援區域估價、攤還表、情境比較、壓力測試與 PDF 匯出。

## 主要功能

- 房貸試算（本息平均攤還）
- 寬限期與每年提前還款模擬
- 區域估價（縣市 / 行政區 / 坪數）
- 行政區快速搜尋篩選
- 新青安分段試算（優惠額度 / 減碼利率 / 優惠年限）
- 情境比較（手動 + 自動生成 3 方案）
- 壓力測試（固定利率 + 蒙地卡羅隨機壓測）
- 視覺化圖表（互動縮放、提示）
- DTI 收支比儀表板（月付 / 月收入風險視覺化）
- PDF 報告匯出（內建引擎，免安裝外部工具）
- UI 模式切換（基本 / 進階）
- 動畫強度切換（高 / 低 / 關）

## 執行環境

- Visual Studio Community 2026
- .NET Framework 4.7.2
- C# 7.3

## 如何執行

1. 使用 Visual Studio 開啟方案。
2. 設定啟動專案為 `1113354_陳冠瑋_房貸計算器`。
3. 按 `F5` 執行。

## 使用流程

1. 輸入房價、自備款、利率、年限、寬限期。
2. （可選）使用區域預設與行政區搜尋快速估價。
3. （可選）勾選 `新青安` 並設定優惠參數。
4. （可選）輸入家庭 `月收入`，觀察 DTI 儀表板。
5. 按 `開始試算`。
6. 可使用 `壓測` / `隨機壓測` 比較風險。
7. 以 `PDF` 匯出完整報告。
